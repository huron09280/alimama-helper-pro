## 4. 关键联动关系（已验证）

1. `营销目标切换` 会导致商品选择丢失，需重新加商品。
2. `选品方式：自定义选品 <-> 好货快投` 切换，也会导致商品重置。
3. `手动出价` 与 `智能出价` 会切换可见子树：
   - 手动出价出现更细分的人群/关键词/创意调价结构；
   - 切回智能出价后收敛为目标化结构。
4. `人群设置` 的父开关控制子层可见与可编辑状态。
5. 高级设置三页签都不是静态展示，均有真实开关/模板/层级交互。

## 6. 插件场景配置复刻映射（关键词推广 / 自定义推广）

以下为“页面最细按钮 -> 插件配置项 -> 提交字段”的一一映射：

1. 营销目标
   - 页面按钮：`搜索卡位` / `趋势明星` / `流量金卡` / `自定义推广`
   - 插件项：`营销目标`（分段按钮）
   - 提交字段：`campaign.promotionScene`、`campaign.itemSelectedMode`、`campaign.bidTargetV2`、`campaign.optimizeTarget`
2. 选品方式
   - 页面按钮：`自定义选品` / `行业推荐选品`
   - 插件项：`选品方式`
   - 提交字段：`campaign.itemSelectedMode`
3. 冷启加速
   - 页面按钮：`开` / `关`
   - 插件项：`冷启加速`
   - 提交字段：`campaign.campaignColdStartVO.coldStartStatus`
4. 人群设置
   - 页面按钮：`智能人群` / `添加种子人群` / `设置优先投放客户` / `关闭`
   - 插件项：`人群设置` + 弹窗 `配置人群`
   - 提交字段：`campaign.needTargetCrowd`、`campaign.aiXiaowanCrowdListSwitch`、`campaign.crowdList`、`adgroup.rightList`
5. 创意设置
   - 页面按钮：`智能` / `专业` / `极简`
   - 插件项：`创意设置`
   - 提交字段：`campaign.creativeSetMode`
6. 高级设置-投放资源位
   - 页面按钮：`全部开启` / `全部关闭` / 行级 `开` `关`
   - 插件项：`投放资源位` + 弹窗 `高级设置`（资源位 Tab）
   - 提交字段：`campaign.adzoneList`
7. 高级设置-投放地域
   - 页面按钮：`当前设置` / `推荐投放地域模板` / `自定义投放地域模板`、`按首字母选择` / `按地理区选择`、`全部地域` / `常用区域`
   - 插件项：`投放地域` + 弹窗 `高级设置`（地域 Tab）
   - 提交字段：`campaign.launchAreaStrList`
8. 高级设置-投放时间
   - 页面按钮：`当前设置` / `全日制投放` / `自定义投放时间模板`、`清空` / `重置`、时间网格单元格
   - 插件项：`投放时间` + 弹窗 `高级设置`（时间 Tab）
   - 提交字段：`campaign.launchPeriodList`
