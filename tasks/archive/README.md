# tasks/archive

本目录存放已完成或历史性的任务流水，避免 `tasks/todo.md` 继续膨胀。归档内容只保留证据和上下文，不代表当前仍在执行。

## 归档文件

- `todo-history-2026-04-15-to-2026-06-14.md`：从旧 `tasks/todo.md` 迁出的历史任务记录，覆盖 2026-04-15 至 2026-06-14 的抓包、授权、UI、复制计划、批量+、内存优化和 AI 点睛等多轮工作。
- `todo-history-2026-06-15-to-2026-06-22.md`：从 `tasks/todo.md` 迁出的已完成收尾任务，覆盖未处理风险收敛、本地主线合并、中文提交与 GitHub 推送记录。

## 使用规则

- 当前会话计划、验证和复盘只写 `tasks/todo.md`。
- 已结束的长流水整体迁入本目录；如仍有明确未完成事项，先提炼到当前 `tasks/todo.md` 的开放风险，再归档历史全文。
- 归档文件中的 `[ ]`、`待补充` 或旧验证提示属于历史记录，不自动等同于当前待办。
