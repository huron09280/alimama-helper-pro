# 关键词推广 -> 趋势明星矩阵页真实运行态验收清单

## 安全边界
- 只能使用 `chrome-devtools` MCP 在真实 `one.alimama.com` 页面验证。
- 验证到本地计划草稿/首页计划列表生成即可，不点击 `立即投放`、`批量创建`、原生 `创建完成` 或其它真实提交入口。
- 结束前检查页面日志或 Network，确认没有触发真实计划创建请求。

## 前置准备
- 运行 `node scripts/build.mjs`，确保根 userscript、packages 和 `dist/extension/` 是当前代码。
- 刷新真实 `one.alimama.com` 页面，确认助手运行态已重新注入。
- 打开 `批量建计划/API 向导`，进入矩阵页，场景为 `关键词推广`。

## 验收步骤
- [ ] 在矩阵页点击 `营销目标=趋势明星`，确认该目标高亮，且编辑页动态配置同步到趋势明星。
- [ ] 确认快捷预设区出现 `选择趋势主题`，点击后新增的维度卡片类型显示为 `选择趋势主题`。
- [ ] 清理该维度后，通过维度卡片尾部 `添加维度` 再次新增，确认自然添加路径同样优先进入 `选择趋势主题`。
- [ ] 在 `选择趋势主题` 维度卡片中，不展开 `点击选择` 下拉即可看到常驻 `添加趋势主题组合`。
- [ ] 展开 `点击选择` 下拉，确认下拉 panel 只负责选择已有值，不包含 `添加趋势主题组合` 新建入口。
- [ ] 点击常驻 `添加趋势主题组合`，打开趋势主题弹窗；选择或保留推荐主题后点 `确定`。
- [ ] 回到矩阵页，确认维度摘要出现 `已选 1 项：已选 N/6：...`，并且下拉中可以勾选刚保存的组合。
- [ ] 在缺商品或缺矩阵配置状态点击 `生成计划`，确认当前矩阵页按钮附近出现红色前置条件失败提示。
- [ ] 回首页添加真实商品，回矩阵页补齐推荐维度，确认摘要显示矩阵开启、组合数和批次数。
- [ ] 点击 `生成计划`，确认本地计划追加到首页计划列表，并看到成功日志；不要继续点击真实投放提交入口。
- [ ] 验证后检查日志/Network，确认未触发真实线上创建计划请求。

## 通过口径
- 目标切换、预设、维度卡片、组合新建、组合选择、生成失败提示和生成成功路径必须在同一刷新后的真实运行态中完成。
- 若账号权限、商品状态或页面变化阻塞某项，需要记录具体阻塞点，不能用本地静态页替代。
